# ConnieHealth Organization Community Health Files

This repository contains organization-wide community health files that apply to all repositories in the ConnieHealth organization by default.

## Pull Request Template

The default PR template is located at `.github/pull_request_template.md` and will be automatically used for all repositories that don't have their own PR template.

Individual repositories can override this template by creating their own `.github/pull_request_template.md` file.

## More Information

For more details about organization-wide community health files, see [GitHub's documentation](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).
